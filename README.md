# OWS Turn Assistant

A small static HTML/CSS/JS helper that walks players through the **Operational Wargame System (OWS)** turn sequence, with a built-in dice roller and per-step local reminders.

## Run locally

- Open `index.html` in your browser (recommended).
- Or serve the folder with any static server and browse to the root.
